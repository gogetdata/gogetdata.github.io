	<h4>Symbol Key:</h4>
	<p><sub><b>Linux: </b><i class="fa fa-linux"></i> - This symbol designates that the package can be installed on an linux system</sub></p>
	<p><sub><b>OSX: </b><i class="fa fa-apple"></i> - This symbol designates that the package can be installed on an OSX system</sub></p>
	<p><sub><b>NOARCH: </b><i class="fa fa-desktop"></i> - This symbol designates that the package was built agnostic to the platform or architecture of the system. If a package is built with noarch it can be assumed that the package can be installed on any platform</sub></p>
	<p><sub><b>Dot: </b><i class="fa fa-dot-circle-o"></i> - This symbol designates that the package can be installed with noarch, but that the package hasn't been designated for the specific platform. (It should work on this platform but has not been designated for the specific platform)</sub></p>
